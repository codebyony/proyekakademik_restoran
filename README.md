## Restoran
Proyek ini adalah aplikasi sederhana untuk membantu restoran mengelola pemesanan meja. Pengguna dapat melihat ketersediaan meja, memesan meja, dan mendapatkan konfirmasi pemesanan.

## Teknologi
PHP, MySQL
